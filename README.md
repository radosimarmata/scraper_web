# Scraper Web
A simple Python script to scrape

## Requirements
- Python 3.x
- `requests` library
- `beautifulsoup4` library

## Installation
1. **Clone the repository:**
```
git clone https://github.com/radosimarmata/scraper_web.git
```
```
cd scraper_web
```
2. **Install requirement**
```
pip install -r requirements.txt
```
